# Lessons

- 2026-06-04: A small OOF lift from threshold/ensemble tuning can regress public score if it changes only a narrow boundary. Record public deltas immediately and inspect prediction transition tables before treating local OOF as the next submission policy.
- 2026-06-04: Targeted second-stage gates on incumbent `STAR`/`QSO` predictions overcorrected and reduced balanced accuracy. Prefer count-based transition diagnostics before productionizing override rules.
- 2026-06-04: Public results now show cached blend gains are real but small: `12_multi_blend` improved the public score to `0.96711`, while `13_class_weight_lgbm` underperformed it at `0.96692`. Future work needs a materially new signal to close the remaining `~0.00289` gap.
- 2026-06-04: **Model convergence is NOT proof of a Bayes-error floor.** I concluded `0.966` was the feature ceiling because 13 diverse models clustered within `0.00075`. That was wrong — they shared a representational blind spot (gradient-boosted trees on raw `alpha`/`delta` can only split axis-aligned, so they cannot encode "fraction of my spatial neighbours that are class c"). The leaderboard (top `0.97127`) was the disconfirming evidence I should have weighted first. **Rule:** before declaring any ceiling, check whether an external reference (leaderboard, published result) contradicts it; if it does, the gap is a technique gap, not a Bayes floor. A k-NN "Bayes estimate" that scores *below* your own models proves the estimator is weak, not that the ceiling is low.
- 2026-06-04: **For synthetic/competition tabular data, test neighbourhood/target-encoding features early.** Class was clustered in sky position (position-only LightGBM `0.678`; 10-NN same-class rate `0.68` vs `0.49` chance). Out-of-fold spatial k-NN class fractions on unit-sphere `(alpha,delta)` lifted OOF from `0.9663` to `~0.969`. The ablation clue was there all along: dropping coordinates cost `0.014`, which is astrophysically implausible unless position encodes the (synthetic) label structure. **Rule:** when a physically-irrelevant feature carries large signal, suspect generative/spatial structure and build neighbourhood target-encoding (with strict OOF + a public-LB transfer gate).
- 2026-06-04: **Public feedback can invalidate local class-bottleneck stories.** Local OOF said GALAXY recall was the wall, but public scores rewarded `19_loo_spatial_final.csv` (`0.96970`) over the more GALAXY-neutral `20_loo_spatial_neutral.csv` (`0.96968`). **Rule:** after a public-risk final-only probe wins, use transition counts from the winning public file, not the prior OOF residual narrative, to choose the next submission direction.
- 2026-06-04: A tied public result is still directional evidence. `23_loo_spatial_star_tilt.csv` tied `19_loo_spatial_final.csv` at `0.96970`, so STAR tilt was safe but not additive. **Rule:** when a tie consumes a probe, pivot to the next independent axis instead of over-sampling the tied axis.
